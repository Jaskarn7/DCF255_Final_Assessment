Reflection

1. With the growth of technology, the size of different devices from which users can access web pages also changed. One can find a huge monitor and open a web page on it but obviously the way things could be seen on the huge monitor will not be same as they be seen on a smartphone. So there must be a way to make the css work for different devices differently respective to their sizes. So the way cones out to be Responsize Web Design or Responsive Design. Responsive Design is an approach through which different content of the web page can be adjusted to different screen sizes

2. Responsive design for a website is mainly created in HTML and CSS file. But there are a lot of technologies presented in the market that can be used to make a webpage responsive. The most popular and effecive one is Gridset. As the name may suggest, it is used to design and build custom and responsive grid based layout.
